What is MAC address?



A media access control address (MAC address) is a unique identifier assigned to a network interface controller (NIC) for use as a network address in communications within a network segment. As typically represented, MAC addresses are recognizable as six groups of two hexadecimal digits, separated by hyphens or colons. This format gives 281,474,976,710,656 possible unique MAC addresses.

IEEE 802 standards define 2 commonly used formats to print a MAC address in hexadecimal digits:

  • Six groups of two hexadecimal digits separated by hyphens (-), like 01-23-45-67-89-ab
  • Six groups of two hexadecimal digits separated by colons (:), like 01:23:45:67:89:ab

Structure of MAC address?

MAC Address is a 48-bits hardware number of a computer, which is embedded into network card (NIC). It has 6 groups of two hexadecimal digits (0-9, a-f). These pair of hexadecimal digits are separated by hyphens (-) or colons (:). Example like 01-23-45-67-89-ab. Following picture shows the internal structure of MAC address:



MAC address is 6 bytes (48 bits), The first 3 bytes (24 bits) represent the manufacturer of the card and the last 3 bytes (24 bits) identify the particular card from that manufacturer.
